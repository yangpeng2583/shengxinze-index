# 山东盛鑫泽科技发展有限公司 - 企业官网

山东盛鑫泽科技发展有限公司官方网站，专注于工业过程在线分析领域。

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 主页（含导航、Banner、关于我们、产品中心、解决方案、企业优势、新闻资讯、联系我们） |
| `cases.html` | 行业案例独立页面（可通过顶部菜单跳转） |
| `images/qrcode_wechat.png` | 微信公众号二维码（请替换为实际图片） |

## 使用说明

1. 将所有文件上传至服务器根目录
2. 将微信公众号二维码图片命名为 `qrcode_wechat.png` 放入 `images/` 目录
3. 直接访问 `index.html` 即可
